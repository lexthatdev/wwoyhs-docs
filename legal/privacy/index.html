privacy policy

updated May 2nd, 2026
this bot is developed by @lex and is designed with minimal data usage in mind.

1. data collection

This bot processes only the data required for its functionality within the Discord server. This may include user IDs, usernames, messages, attachments, and interaction data necessary for features such as moderation, verification and command handling.

2. data storage

Some data may be stored temporarily to support bot features (for example, moderation actions, verification processees, or configuration data). Data is not retained longer than necessary for its intended purpose.

3. data usage

Collected data is used exclusively to operate and improve the bot's functionality. Data is not sold, rented or shared with any third parties.

4. data access

Access to stored data is limited to the bot and its developer only. Data is not exposed publicly except where required for bot functionality (e.g., messages sent within Discord channels).

5. security

Reasonable measures are taken to protect data handled by the bot. However, no system can be guaranteed completely secure, and users acknowledge this inherent risk when interacting with the bot.

6. third-party services

This bot operates on the Discord platform and relies on external hosting and infrastructure. As such, data may also be subject to Discord's own policies and terms.

7. user rights

Users may request the removal of any stored data relating to them by contacting the developer. Requests will be handled within a reasonable timeframe.

8. changes to this policy

This Privacy Policy may be updated at any time to reflect changes in the bot's functionality or data handling practices. Continued use of the bot constitutes acceptance of these changes.
