privacy policy

updated May 8th, 2026
this bot is developed by @lex and is designed with minimal data usage in mind.

1. data collection

This bot processes data required for its functionality within the Discord server. This includes user IDs, usernames, messages, attachments, and interaction data necessary for features such as moderation, verification, analytics, and command handling.

2. data storage

The bot stores structured data locally, including user activity logs (such as message counts, edits, and deletions) and moderation logs (such as bans, kicks, and warnings). Data is stored only as long as needed for bot functionality and server operations.

3. data usage

Collected data is used exclusively to operate, improve, and support bot features including moderation tools, user logs, and server analytics. Data is not sold, rented, or shared with third parties.

4. data access

Stored data is accessible only to the bot and its developer. Certain aggregated or contextual data may be displayed within Discord through bot commands (e.g., /logs or /modlogs).

5. security

Reasonable measures are taken to protect stored data. However, no system can be guaranteed completely secure, and users acknowledge this risk when interacting with the bot.

6. third-party services

This bot operates on Discord and relies on external hosting infrastructure. Data may also be subject to Discord’s own terms and privacy policies.

7. user rights

Users may request removal of stored data related to them by contacting the developer. Requests will be handled within a reasonable timeframe where technically possible.

8. changes to this policy

This Privacy Policy may be updated at any time to reflect changes in functionality or data handling. Continued use of the bot constitutes acceptance of these changes.
